Mouse

Botões Básicos
Name Description
{LButton} Botão esquerdo do mouse.
{RButton} Botão direito do mouse.
{MButton} Botão do scroll do mouse.
Botões Avançados
Name Description
{XButton1} 4º botão do mouse. Normalmente age como retornar em browsers.
{XButton2} 5º botão do mouse. Normalmente age como avançar em browsers.
Roda
Name Description
{WheelDown} Gira o scroll para baixo.
{WheelUp} Gira o scroll para cima.
{WheelLeft}
{WheelRight}

[v1.0.48+]: Scroll to the left or right.

Gira o scroll horizontalmente para esquerda ou direita.
